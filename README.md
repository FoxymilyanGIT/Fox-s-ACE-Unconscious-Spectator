# Fox's ACE Unconscious Spectator

A small Arma 3 addon for ACE Medical that opens ACE Spectator when the local player becomes unconscious.

## Features

* Opens ACE Spectator on ACE unconscious state.
* Restricts spectatable players to the same side.
* Keeps the unconscious player's body visible and treatable.
* Closes spectator on wake-up, death, and respawn.

## Requirements

* Arma 3
* CBA_A3
* ACE3
* HEMTT for building

## Building

Check the project:

```powershell
hemtt check
```

Build development version:

```powershell
hemtt dev
```

Build release version:

```powershell
hemtt release
```
